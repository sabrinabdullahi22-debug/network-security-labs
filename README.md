# network-security-labs
Syftet med detta projekt är att stärka mina färdigheter inom:

VLAN-segmentering

Inter-VLAN-routing

Åtkomstkontrollistor (ACL)

Säker enhetskonfiguration (SSH, portsäkerhet)

Hotmodellering och riskreducering

Alla laborationer är byggda med Cisco Packet Tracer och VMware i en kontrollerad labbmiljö.

Varje laboration innehåller:

Topologidiagram

Konfigurationsfiler

Säkerhetsanalys

Lärdomar

Det är där skillnaden ligger mellan att “få det att funka” och att faktiskt tänka som en systemarkitekt. Funktionalitet är minimumkravet. Säkerhet och dokumentation är det som gör att infrastrukturen överlever verkligheten – med användare, misstag och illvilliga aktörer inkluderade.
